# Stage2.5 Archival Note

## Status

Stage2.5 is archived as an exploratory, non-authoritative side-path.
It is not part of the active benchmark pipeline as of 2026-03-25.

## What Stage2.5 Attempted

- recover component-aware structure from frozen Stage2 formulation rows
- build text-first evidence packs for harder component reconstruction cases
- explore a shadow-only recovery path without changing Stage3 or Stage5

## Why It Is Not Mainline

- the project has discontinued Stage2.5 as the mainline patching strategy
- the active benchmark pipeline must remain a clean Stage2 -> Stage3 -> Stage5 path
- the current direction is a fuller Stage2 redesign rather than continued
  patching through a Stage2.5 side-path

## What Remains Useful

- evidence-pack experiments remain useful as design input for later
  deterministic evidence binding
- component-shadow validation runs remain useful for failure examples and
  schema redesign constraints
- Stage2.5 code remains available for historical comparison and method review

## Non-Change Statement

No active pipeline code was changed by this note itself.
This note records archival status only.
